## 快速工作流

```
1. 打开 Leonardo.ai → AI Image Generation
2. 选模型：Leonardo Lightning XL（最快）或 Photography 模式
3. 每次复制上方提示词，尺寸选 1:1 (1024×1024)
4. 每款产品生成4张，挑最好的1张
5. 下载 → 命名为 product-XX.webp → 放到 images/ 文件夹
6. 告诉我，我帮你更新 HTML
```
