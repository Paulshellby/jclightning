# JC Lightning 网站背景图 · AI 生图提示词 V2

## 为什么重做？

V1 所有深色区背景图都用纯黑底 `#0e0c0a`，太暗——文字（白/金）虽然对比度够，但整体页面给人沉重、压抑的感觉。Hero 区尤其严重：纯黑底 + 仅 CSS 渐变，hero 标语虽然是白色大字，但周围一片漆黑，字体感觉"看不清"。

## V2 改动

- 底色调亮：纯黑 `#0e0c0a` → 炭灰 `#1e1a14` ~ `#24201c`
- 金色环境光增强：从几乎看不见（5-8%）→ 明显但克制的暖光（10-18%）
- Hero 文字区加暖光：左侧（文字）区域有柔和金色光晕，右侧保持暗调衬托产品图
- 深色区整体"呼吸感"：图案更清晰、光点更明显

## 生成后操作

1. 图片放入：`D:\嘉晨灯饰JC Lightning\网站\images\`
2. 重命名为上表文件名（如 `bg-hero.jpg`）
3. **Hero CSS 需要同步修改** —— 见下方说明
4. 告诉我，我帮你一次性写入 CSS

### Hero CSS 修改说明（生成图片后必须改）

当前 Hero 只有纯黑 CSS 背景，没有图片：
```css
.hero { background:var(--ink); }
```

生成 `bg-hero.jpg` 后，需要改为：
```css
.hero { background:var(--ink) url(images/bg-hero.jpg) center/cover no-repeat; }
```

同时 `.hero-bg` 的金色光晕可以降低（因为图片里已有暖光），`.hero-grain` 保持。

## 注意事项

- 所有图片透明度 10-18%（V1 是 5-12%，V2 更明显但不抢文字）
- 色系：暖炭灰底色 + 铜金/亮金色点缀（#a87d2a / #c9a24e）
- 格式：JPG，每张 100-300KB
- 推荐工具：Midjourney（加 `--style raw --ar 16:9 --no text,logo,watermark`）或 DALL·E 3
