## 快速工作流

```
1. 打开 Leonardo.ai → AI Image Generation
2. 选择模型：Leonardo Lightning XL（速度快）或 Photography（更真实）
3. Style Preset: "Product Photography" 或 "Cinematic"
4. 复制上方提示词，粘贴到 Prompt 框
5. 尺寸选 1:1 (1024×1024)
6. Negative Prompt 粘贴上方的通用负面提示词
7. 每款产品生成 4 张，挑 1 张最好的
8. 下载 → 命名为 product-XX.webp → 放到 images/ 文件夹
9. 告诉我，我帮你更新 HTML
```

## 提示词使用技巧

- **白底图**：适合作为产品列表封面，干净统一
- **场景图**：适合 hero 大图、社媒帖子、目录册封面
- **Leonardo.ai**：建议开启 `Alchemy` 增强画质，`Prompt Magic` 强度设为 0.3-0.5
- **DALL-E 3**：提示词可直接使用，注意加 `--no text watermark`
- 如果生成结果有文字/水印 → 加强 Negative Prompt
- 如果产品形态不准确 → 在提示词前面加详细的形态描述
